Lab 13: Typography I

Challenge

In this lab, we were asked to create a lab 13 folder complete with the usual index.html file, css & img folders, and organize our page accordingly. We were then asked to create a frame and style it. We were then asked to pick a website and replicate the typography on our own site and link it using a second created html file.
